# BOOK Project

## Techs Stack

### Backend

- **JAVA**
- **Maven**
- **Spring Boot**
- **Spring Data JPA**
- **Spring Security**

### Fontend

- **React**
- **Vite**
- **Redux**
- **Redux Persist**
- **Tailwindcss**
- **Axios**

### Run Project(Development)

`api`:

```sh
./mvnw spring-boot:run
```

`client`:

```sh
yarn
yarn dev
```
